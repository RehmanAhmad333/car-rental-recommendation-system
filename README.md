# Car Rental Recommendation System

## Project Overview
An AI-powered recommendation engine that suggests cars based on car similarity, user preferences, and booking history. The system combines multiple filtering techniques to deliver personalized and relevant car rental options.

## Goal
Build a robust recommendation system that provides accurate car suggestions by leveraging:
- Content-Based Filtering
- Collaborative Filtering
- Hybrid Recommendation Approaches

## Tech Stack
- **Python** – Core programming language
- **Scikit-learn** – Machine learning algorithms for similarity computation
- **FastAPI** – Lightweight web framework to serve the recommendation API
- **MongoDB** – NoSQL database for storing car details, user profiles, and booking history
